# Report Generator LangGraph Flow 
A LangGraph based project that can generate report on any topic with references<br>
This project involves tools like:\
1- Tavily Search for Searching reliable sources for the given topic.\
2- Trafilatura for extracting text from the sources.\
3- Groq LLM for Generating a concise report on the topic.\
The Workflow of the graph is shown below <br>
This app is deployed on Streamlit.\
